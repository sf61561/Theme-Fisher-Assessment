# Portfolio Website - Daryl Smith

## Project Overview

This is a modern, responsive portfolio website for Daryl Smith, a UI/UX Designer and Creative Developer. The site showcases professional background, education, work history, and skills with a sleek, dark-themed design. The portfolio features a clean navigation system, an interactive sidebar menu, and sections for introduction, education, and work experience.

## Tech Stack Used

- **HTML5** - Semantic markup and structure
- **CSS3** - Custom styling with font families and animations
- **Tailwind CSS** - Utility-first CSS framework (via CDN)

## Steps to Run the Project Locally

1. **Clone or download the project**
   ```bash
   git clone <repository-url>
   cd Theme Fisher Assessment
   ```

2. **Open the project directory**
   ```bash
   cd e:\Theme Fisher Assessment
   ```

3. **Serve the project**
     Simply open `index.html` in your web browser by double-clicking the file.

4. **View the website**
   - Navigate to the local server URL in your web browser
   - The site is fully responsive and works on desktop, tablet, and mobile devices
